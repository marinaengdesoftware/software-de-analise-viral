
# Pipeline Geral do Sistema

O sistema foi projetado para realizar engenharia reversa de conteúdos virais
através da extração e análise multimodal de dados.

Etapas principais:

1. Ingestão de vídeo
2. Segmentação de cenas
3. Extração de frames relevantes
4. Análise de movimento
5. Extração de áudio
6. Análise semântica
7. Identificação de beats narrativos
8. Modelagem de padrões emocionais
